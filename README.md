# MahsaNG v16

MahsaNG version 16 - Android Application

## نسخه‌ها (Versions)

### v16

آخرین نسخه از برنامه MahsaNG

## فایل‌های موجود

- `MahsaNG_16_arm64-v8a.apk` - برنامه برای دستگاه‌های ARM 64-bit

## نحوه نصب (Installation)

1. فایل APK را دانلود کنید
2. روی دستگاه اندروید خود فایل را باز کنید
3. مراحل نصب را دنبال کنید

## توجهات امنیتی (Security)

مطمئن شوید که:
- از منابع موثوق دانلود کنید
- دستگاه شما از منابع نامعلوم نصب برنامه‌ها را مجاز کرده باشد

## درباره MahsaNG

MahsaNG یک پروژه آزاد است که برای استفاده عمومی طراحی شده است.
